# For Abinaya | Valentine's Day 2026

An elegant, sophisticated website expressing love on Valentine's Day.

## Design Philosophy

This website embodies elegance and sophistication through:
- **Minimalist Design** - Clean lines and purposeful whitespace
- **Premium Typography** - Playfair Display, Cormorant Garamond, and Montserrat
- **Refined Color Palette** - Warm earth tones with elegant accents
- **Smooth Interactions** - Subtle animations and transitions
- **Professional Layout** - Grid-based, responsive design

## Structure

### Five Elegant Sections:

1. **Hero** - Personalized greeting with Abinaya's name prominently displayed
2. **Opening** - Sincere expression of feelings and significance
3. **Qualities** - Six attributes that make her special (grid layout)
4. **Promises** - Five commitments for the future together
5. **Final Message** - Heartfelt closing with signature

## Features

- **Scroll Navigation** - Navigate between sections with smooth scrolling
- **Navigation Dots** - Right-side indicators showing current section
- **Keyboard Support** - Arrow keys, Page Up/Down, Home, End navigation
- **Parallax Effects** - Subtle depth on hero section
- **Intersection Animations** - Elements fade in as you scroll
- **Custom Cursor** - Elegant cursor effect (desktop only)
- **Fully Responsive** - Optimized for desktop, tablet, and mobile

## How to Use

### Local Viewing:
1. Open terminal in the project folder
2. Run: `python -m http.server 8000`
3. Visit: `http://localhost:8000`

### Direct Opening:
- Double-click `index.html` to open in your browser

### Online Deployment:
Deploy to any static hosting service:
- **GitHub Pages** - Free hosting with custom domain support
- **Netlify** - Drag-and-drop deployment
- **Vercel** - Automatic HTTPS and global CDN
- **Cloudflare Pages** - Fast and secure hosting

## Customization

Edit `index.html` to personalize:

- **Section 2** - Opening messages about your relationship
- **Section 3** - Six qualities (currently: Grace, Kindness, Strength, Intelligence, Love, Presence)
- **Section 4** - Five promises (customize to your commitment)
- **Section 5** - Final message and signature

### Changing Colors:
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2c1810;
    --accent-color: #c9a17c;
}
```
